User Input & Forms

Have you heard the term IU being thrown around? Well, this isn't it. This is User Input. User input are the features like text fields, radio buttons, and check boxes. Here is a sample form I made a long while back.

Text Field

<input type="text" id="f_name" name="firstname" required >
<button type="submit">
submit
</button>
Creates a text field.
name="firstname"
required makes the form not move on until it has anything here.
